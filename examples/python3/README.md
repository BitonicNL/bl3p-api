Unofficial Python 3 Bl3p client exchange api
===

## Install
```
$ git clone https://github.com/joosthoeks/bl3p-api.git
$ cd bl3p-api/examples/python3
$ [sudo] pip3 install -e .
```

## Update
```
$ cd bl3p-api
$ git pull
```

## Test
```
$ cd examples/python3/
```
modify public_key and secret_key in example.py then run:
```
$ python3 example.py
```
